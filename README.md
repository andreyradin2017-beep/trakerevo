## 🏗 Как запустить локально

1. **Установка зависимостей**:

   ```bash
   npm install
   ```

2. **Запуск разработки**:

   ```bash
   npm run dev
   ```

3. **Проверка стабильности**:
   ```bash
   npm run build  # Сборка и проверка типов
   npm run test   # Запуск всех тестов
   ```
